# Backend-Training

This repository is designed to help you learn and practice **Backend Development** using **Node.js**, **Express.js**, and essential tools and concepts. Each folder contains a hands-on assignment or concept with its own working code.

---

## Steps to Create and Use This Repository

### 1. Create Repository

- Go to GitHub.
- Create a new repository named **`backend-training`**.
- Initialize it with a `README.md`.

### 2. Clone the Repository

Open terminal and run:
```bash
git clone https://github.com/<your-username>/backend-training.git
cd backend-training
